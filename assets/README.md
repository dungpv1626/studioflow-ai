# Studio Flow — Brand Assets

## Cấu trúc thư mục

```
assets/
├── logo/
│   ├── logo_primary.png       # Logo màu chính (nền trắng/sáng)
│   ├── logo_white.png         # Logo trắng (nền tối/xanh)
│   ├── logo_icon.png          # Icon vuông (dùng cho avatar, favicon)
│   └── logo_horizontal.png   # Logo nằm ngang
├── templates/
│   ├── facebook_post.png      # Template bài đăng Facebook (1080x1080)
│   ├── facebook_story.png     # Template Story (1080x1920)
│   ├── hero_banner.png        # Banner trang chủ (1920x600)
│   ├── ad_square.png          # Quảng cáo vuông (1080x1080)
│   └── email_header.png       # Header email
└── infographics/
    ├── feature_invoice.png    # Tính năng hóa đơn
    ├── feature_calendar.png   # Tính năng lịch hẹn
    ├── feature_report.png     # Tính năng báo cáo
    ├── pricing.png            # Bảng giá 3 gói
    └── how_it_works.png       # Sơ đồ cách hoạt động
```

## Quy tắc đặt tên
- Dùng chữ thường, dấu gạch dưới, không dấu tiếng Việt
- Format: PNG (logo, template) hoặc JPG (ảnh thực tế)
